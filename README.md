# Haseeb Portfolio — Skills Update

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Contact form activation

The contact form submits through FormSubmit to `haseebmujeeb360@gmail.com`.
The first test submission triggers an activation email. Open that email and approve the form once; future enquiries will then be forwarded to the inbox.
